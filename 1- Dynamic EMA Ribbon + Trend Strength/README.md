Dynamic EMA Ribbon Indicator
Pine Script v5 – Trend identification tool for TradingView.

Description:
Plots 3 EMAs (20, 50, 100) as a ribbon:

Bullish: EMA20 > EMA50 > EMA100 → green background

Bearish: EMA20 < EMA50 < EMA100 → red background

Alerts for EMA20/EMA50 crossovers included

Usage:

Add to chart and watch background colors for trend.

Use price action signals as entry confirmation: test EMA20 with trend alignment.

Alerts are confirmation only, not standalone entry signals.

Notes:

Backtest before live trading.

Can combine with other filters or multi-timeframe analysis for accuracy.
