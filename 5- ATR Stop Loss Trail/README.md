ATR Trail Stop Indicator
Pine Script v5 – Dynamic trailing stop based on ATR.

Description:
Plots trailing stops for long (green) and short (red) positions using ATR(14) × 2:

Long Stop: Close − 2×ATR (acts as support)

Short Stop: Close + 2×ATR (acts as resistance)

Usage:

Add the indicator to TradingView.

Use green line to trail long positions, red line for short positions.

Adjust ATR period or multiplier in code if needed.

Notes:

For educational use; backtest before live trading.

Can combine with other strategy filters for better risk management.
